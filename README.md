# Job Connections Project Repo

#### (1) Clone Repo
Clone the repository to your local machine:

`git clone https://github.com/porterolson/jcp.git`

#### (2) Download Anaconda
Download Anaconda if you do not have it already:

https://www.anaconda.com/download
